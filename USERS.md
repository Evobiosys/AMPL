## Projects using AMPL indirectly (via the EvoBiosys License)

These projects adopt the EvoBiosys License, which is itself built on AMPL —
so they comply with AMPL 1.0 as their base license, plus the EvoBiosys
addendum on top.

- [evobiosys.org](https://github.com/Evobiosys/evobiosys.org)
